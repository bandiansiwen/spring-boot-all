# 动态规划

### 什么是动态规划？

动态规划是一种通过将复杂问题分解为更简单的子问题来求解的方法。它适用于具有重叠子问题和最优子结构的问题。

### 动态规划的核心思想

1. 记忆化：存储子问题的解，避免重复计算 
2. 状态转移：定义状态和状态转移方程 
3. 自底向上：从小问题开始，逐步解决大问题

### 动态规划解题步骤

1. 定义状态：定义子问题，通常使用一个数组（一维、二维或多维）来保存子问题的解。
2. 确定状态转移方程：找出状态之间的关系，即如何通过已知状态推导出未知状态。 
3. 初始化边界条件：确定最基本、最小的子问题的解。 
4. 计算顺序：确定计算状态的顺序，通常是自底向上（递推）或自顶向下（记忆化搜索）。

### 动态规划类型

1. 线性DP：斐波那契、爬楼梯 
2. 区间DP：矩阵连乘、石子合并 
3. 背包DP：0-1背包、完全背包 
4. 树形DP：在树结构上的DP 
5. 状态压缩DP：用位运算表示状态

### 解题模板

```java
public int dpSolution(参数) {
    // 1. 定义状态数组
    int[] dp = new int[n];
    
    // 2. 初始化边界条件
    dp[0] = ...;
    
    // 3. 状态转移
    for (int i = 1; i < n; i++) {
        dp[i] = ... // 根据状态转移方程计算
    }
    
    // 4. 返回结果
    return dp[n-1];
}
```
