# 贪心算法

### 什么是贪心算法？

贪心算法是一种在每一步选择中都采取当前状态下最优（即最有利）的选择，从而希望导致结果是全局最优的算法策略。

### 贪心算法的核心思想

**"局部最优 → 全局最优"**：
- 每一步都做出在当前状态下最好的选择
- 不考虑长远影响，只关注眼前利益
- 希望局部最优选择能导致全局最优解

### 贪心算法的步骤

1. 将问题分解为若干个子问题。
2. 对每一个子问题求解，得到子问题的局部最优解。
3. 把子问题的解合并成原问题的一个解。

### 贪心算法的适用条件

1. 贪心选择性质：局部最优选择能导致全局最优解 
2. 最优子结构：问题的最优解包含子问题的最优解

### 贪心算法的关键要点

1. 贪心选择性质：证明局部最优能导致全局最优 
2. 无后效性：当前选择不会影响后续选择的最优性 
3. 贪心策略：确定每一步的最优选择标准

### 贪心算法的证明技巧

1. 替换法：证明任何最优解都可以转换为贪心解 
2. 归纳法：证明贪心选择在每一步都是最优的 
3. 反证法：假设贪心解不是最优，导出矛盾
